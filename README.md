# PaintBall-com-API-completa
O site PaintBall foi desenvolvido com o objetivo de criar uma plataforma de aluguel, aplicada ao contexto de paintball. O sistema utiliza Python com Flask no back-end, API integrada, SQLAlchemy e SQL Server para banco de dados. No front-end, foram usados HTML, CSS e JavaScript, garantindo funcionalidade, organização e interação com o usuário
